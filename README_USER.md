# Инструкция по установке Nova Domains Count

Это расширение позволяет мгновенно подсчитывать количество запросов к доменам на открытой вкладке.

## Как установить расширение

1. **Скачайте и распакуйте архив**:
   - Распакуйте файл `Nova_Domains_Count.zip` в любую удобную папку (например, на Рабочий стол). Вы должны увидеть внутри папку `dist`.

2. **Откройте настройки расширений в Chrome**:
   - Введите в адресной строке браузера: `chrome://extensions/`
   - Или нажмите на иконку пазла (Расширения) -> "Управление расширениями".

3. **Включите Режим разработчика**:
   - В правом верхнем углу страницы переключите тумблер **"Режим разработчика"** (Developer mode).

4. **Загрузите расширение**:
   - Нажмите на кнопку **"Загрузить распакованное расширение"** (Load unpacked) в левом верхнем углу.
   - Выберите папку `dist` из распакованного архива.

5. **Готово!**:
   - Расширение появится в списке. Нажмите на иконку пазла в браузере и "закрепите" (Pin) **Nova Domains Count**, чтобы оно всегда было под рукой.
