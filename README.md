# Nova Domains Count

**Nova Domains Count** — это бесплатное расширение для Chromium-браузеров (Chrome, Edge, Brave и др.) с открытым исходным кодом, предназначенное для мониторинга сетевой активности на веб-страницах.

## Основные возможности

*   **Мгновенный перехват**: Отслеживает запросы (Fetch, XHR, Beacon) в момент их инициации, не дожидаясь ответа сервера.
*   **Обвновления**: Регистрирует все запросы к доменам в режиме реального времени.
*   **Группировка по доменам**: Автоматически выделяет корневые домены и их поддомены для удобного анализа.
*   **Точный счетчик**: Считает каждое обращение к ресурсу, помогая выявить скрытую активность трекеров или рекламных сетей.
*   **Dark Mode**: Полноценная поддержка темной и светлой тем с сохранением выбора.
*   **Экспорт**: Возможность копирования всех найденных доменов одним кликом или одного конкретного домена.

*   **Минимализм**: Легкий интерфейс без лишних деталей и назойливых уведомлений. Отображаются только верхние домены.

## Установка

1.  Скачайте актуальный архив из [релиза](https://github.com/confeden/Nova-Domains-Count/releases).
2.  Распакуйте архив в удобное место.
3.  Перейдите в `chrome://extensions/`.
4.  Включите **"Режим разработчика"** (Developer mode).
5.  Нажмите **"Загрузить распакованное расширение"** (Load unpacked) и выберите распакованную папку `Nova_Domains_Count`.

## Безопасность и Конфиденциальность

Расширение работает полностью локально в вашем браузере. Оно не собирает, не хранит и не передает никакие данные на внешние серверы. Весь перехват происходит внутри контекста страницы исключительно для целей отображения статистики в Popup-окне.
